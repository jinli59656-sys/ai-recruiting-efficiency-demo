# 高级Java后端开发工程师测试岗位

## 岗位名称
高级Java后端开发工程师

## 岗位职责
- 负责企业级后端系统设计、开发与优化
- 参与微服务架构设计与拆分，推进系统高可用与高并发改造
- 负责核心业务接口开发，保证系统稳定性与性能
- 参与数据库设计与 SQL 优化
- 与前端、测试、产品协作推进项目交付

## 任职要求
- 3年以上 Java 后端开发经验
- 熟悉 Spring Boot、Spring Cloud、MySQL、Redis
- 熟悉 Kafka 或 RabbitMQ 等消息队列
- 有微服务、高并发、分布式系统经验优先
- 熟悉 Linux、Docker、Git 等开发运维工具
- 具备良好的沟通协作能力和问题分析能力

## 筛选建议
- 最低学历：本科
- 最少工作年限：3年
- 必须技能：Java、Spring Boot、MySQL
