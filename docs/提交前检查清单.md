## Demo 验证

1. 启动后端：

```powershell
cd backend\app
..\venv\Scripts\uvicorn.exe main:app --reload --host 0.0.0.0 --port 8000
```

2. 启动前端：

```powershell
cd front
npm.cmd run dev -- --host 0.0.0.0
```

3. 打开 `http://localhost:5173`
4. 点击“演示模式登录”
5. 进入“数据采集”
6. 选择示例消息
7. 点击“AI 解析并同步”
8. 确认抽取结果出现
9. 确认腾讯文档镜像表新增一行
10. 进入“工作台”，确认统计和最近动态更新
